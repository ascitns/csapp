## 3.04 访问信息

### 3.04.1 操作数指示符

- 寻址格式**偏移量(基址寄存器, 变址寄存器, 比例因子)**，写成通用表示就是** `Imm(rb, ri, s)`**

- **Imm** 是一个立即数偏移量，可以为空（默认为 0）
- **rb** 是基址寄存器，任意 64 位寄存器都可以，可以为空
- **ri** 是变址寄存器，**除 `%rsp` 外的**任意 64 位寄存器，可以为空
- **s** 是比例因子，只能是 **1、2、4 或 8**，对应 C 中 `char`、`short`、`int/float`、`long/double/指针` 的大小（数组求偏移量用的多）
- 书上给出的表格包含以下变式，均从上述通用形式派生：

| 形式 | 示例 | 实际访问的地址 |
| :--- | :--- | :--- |
| 绝对地址 | `0x100` | `0x100` |
| 间接寻址 | `(%rax)` | `R[%rax]` |
| 基址+偏移 | `8(%rax)` | `8 + R[%rax]` |
| 变址寻址 | `(%rax,%rcx)` | `R[%rax] + R[%rcx]` |
| 变址+缩放 | `(%rax,%rcx,4)` | `R[%rax] + R[%rcx]×4` |
| 完整形式 | `8(%rax,%rcx,4)` | `8 + R[%rax] + R[%rcx]×4` |

### 3.04.2 数据传送指令

*mov的笔记做到上一节了*

- **没有 `movzlq`**，因为把 32 位值零扩展到 64 位时，编译器直接生成一条 `movl`

### 3.04.3 数据传送示例

十六个通用寄存器：

| 64 位 | 32 位 | 16 位 | 8 位 | 惯用角色 |
| :--- | :--- | :--- | :--- | :--- |
| `%rax` | `%eax` | `%ax` | `%al` | 函数**返回值** |
| `%rbx` | `%ebx` | `%bx` | `%bl` | **被调用者保存** |
| `%rcx` | `%ecx` | `%cx` | `%cl` | 第 4 个函数参数 |
| `%rdx` | `%edx` | `%dx` | `%dl` | 第 3 个函数参数 |
| `%rsi` | `%esi` | `%si` | `%sil` | 第 2 个函数参数 |
| `%rdi` | `%edi` | `%di` | `%dil` | 第 1 个函数参数 |
| `%rbp` | `%ebp` | `%bp` | `%bpl` | **被调用者保存** / 帧指针 |
| `%rsp` | `%esp` | `%sp` | `%spl` | **栈指针**（不可随意改写） |
| `%r8` | `%r8d` | `%r8w` | `%r8b` | 第 5 个函数参数 |
| `%r9` | `%r9d` | `%r9w` | `%r9b` | 第 6 个函数参数 |
| `%r10` | `%r10d` | `%r10w` | `%r10b` | 调用者保存 |
| `%r11` | `%r11d` | `%r11w` | `%r11b` | 调用者保存 |
| `%r12` | `%r12d` | `%r12w` | `%r12b` | **被调用者保存** |
| `%r13` | `%r13d` | `%r13w` | `%r13b` | **被调用者保存** |
| `%r14` | `%r14d` | `%r14w` | `%r14b` | **被调用者保存** |
| `%r15` | `%r15d` | `%r15w` | `%r15b` | **被调用者保存** |

- 老寄存器的子寄存器命名遵循规律：64 位用 `%r` 前缀，32 位用同名 `%e` 前缀，16 位去掉前缀字母，8 位用 `l` 后缀（如 `%al`、`%sil`）。
- 新增的 `%r8` 到 `%r15` 统一用后缀表示宽度：`d` 是 32 位双字，`w` 是 16 位字，`b` 是 8 位字节。

### 3.04.4 压入和弹出栈数据

- **栈**在 x86-64 中是内存里的一块连续区域，**向低地址方向增长**
- 寄存器 `%rsp` 始终保存**栈顶的地址**——当前栈上最后一个有效数据的所在位置

向栈中压入数据和弹出数据各有一条专用指令：

```asm
pushq   S       # 等价于: subq $8, %rsp ; movq S, (%rsp)
popq    D       # 等价于: movq (%rsp), D ; addq $8, %rsp
```
